----------board game forum------------

1.each user has a profile where he can upload:
-the board games he has
-the board games he wishes to have
-photos in a gallery
-friends & messages

2.each board game has it's own page:
-a description
-gallery
-reviews & ranking
-users can rate the difficulty / optimal number of players
-link to a buy page
-board game ranking, depending on user scores

3.there's also a retail marketplace implemented, where users can buy:
-board games
-accessories for board games:dice, inserts, 
-merch
